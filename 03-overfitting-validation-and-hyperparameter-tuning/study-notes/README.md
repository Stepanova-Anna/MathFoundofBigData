# Study Notes (обязательная часть самостоятельной работы)

Эта папка используется для углубленных заметок по переобучению,
валидации и подбору гиперпараметров, которые применяются в ЛР 03.

Отдельно обязателен файл `glossary.md`:
- пополняйте его по ходу выполнения каждого ноутбука;
- добавляйте незнакомые термины с простыми объяснениями и источниками.

Когда использовать:
- если нужно отдельно зафиксировать разницу между `train`, `validation`, `test`;
- если вы хотите сравнить несколько candidate feature set на одном и том же split;
- если нужно подробно пояснить, почему модель с более высокой train-метрикой не обязательно лучше;
- если вы хотите разобрать, почему `Pipeline` полезен только тогда, когда все обучаемые preprocessing steps реально живут внутри CV.

Минимальные требования к каждой заметке:
1. Кратко указать, **что изучено**.
2. Пояснить, **где это применимо** в текущей ЛР.
3. Дать **сравнение с альтернативой**.
4. Отразить **ограничения/риски** подхода.
5. Указать **источники** (1-2+).

- `train-validation-test-split.md`
- `gridsearchcv-practice.md`
