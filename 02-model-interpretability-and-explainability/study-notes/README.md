# Study Notes (обязательная часть самостоятельной работы)

Эта папка используется для углубленных заметок по методам интерпретации,
которые применяются в ЛР 02.

Отдельно обязателен файл `glossary.md`:
- пополняйте его по ходу выполнения каждого ноутбука;
- добавляйте незнакомые термины с простыми объяснениями и источниками.

Когда использовать:
- если нужно отдельно зафиксировать ограничения метода интерпретации;
- если вы сравниваете два способа объяснения одной и той же модели;
- если хотите подробно разобрать риск неверной интерпретации.

Минимальные требования к каждой заметке:
1. Кратко указать, **что изучено**.
2. Пояснить, **где это применимо** в текущей ЛР.
3. Дать **сравнение с альтернативой**.
4. Отразить **ограничения/риски** метода.
5. Указать **источники** (1-2+).
