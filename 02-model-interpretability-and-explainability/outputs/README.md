# Outputs

Сюда ноутбуки сохраняют промежуточные и итоговые таблицы:
- `global_importance_comparison.csv` — глобальные ранжирования признаков для разных моделей и методов.
- `partial_dependence_summary.csv` — компактная сводка one-way partial dependence.
- `error_case_explanations.csv` — локальные объяснения наиболее показательных ошибок.

Локально в ноутбуках также могут формироваться вспомогательные DataFrame
для визуализации и narrative-анализа, но в git они не требуются.
